# Flaggen

Die SVG-Flaggen in diesem Ordner stammen aus dem Projekt
**[flag-icons](https://github.com/lipis/flag-icons)** von Panayiotis Lipiridis.

- Lizenz: **MIT** (Quellcode/Set). Die abgebildeten Nationalflaggen sind
  gemeinfrei (public domain).
- Format: 4:3-SVG, lokal eingebunden – es werden **keine** externen Server zur
  Laufzeit kontaktiert (datenschutzfreundlich).

Die Dateinamen entsprechen dem ISO-3166-Alpha-2-Code (z. B. `de.svg`,
`fr.svg`). Für die britischen Landesteile werden die Sondercodes
`gb-eng.svg`, `gb-sct.svg`, `gb-wls.svg` und `gb-nir.svg` verwendet.

Die Zuordnung Mannschaft → ISO-Code steht in `src/Data/teams.php` (Feld `iso2`).
Neue Flaggen können einfach als weitere `<iso2>.svg` hier abgelegt werden.
